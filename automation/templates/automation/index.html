Automation Controls

Getting Started

  1. Create Automators which are capable of performing the operations you want to automate
  2. Determine the conditions under which you want the automator to be called
  3. Setup deciders for each of the conditions under 2.
  4. Setup a controller with the deciders and automators determined above

You can reuse any deciders or automators you create to perform other decisions, so don't be too specific when creating deciders or Automators
